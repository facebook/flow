This post announced indexed access types when it was introduced. For current syntax and behavior, see the [Indexed Access Types](/en/docs/types/indexed-access) documentation.
:::

Flow’s Indexed Access Types are a new type annotation syntax that allows you to get the type of a property from an object, array, or tuple type.
